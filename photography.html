WILDLIFE PHOTOGRAPHY PORTFOLIO

UI FOR THE HERO SECTION OF A WILDLIFE PHOTOGRAPHY PORTFOLIO:

DESCRIPTION

This is a fictional project I designed for a web design course on Udemy.


The idea behind this hero section of a web design is to create a portfolio for wildlife photographers, where they can showcase their photos, videos, and stories on the web.


MY ROLE

COLOR PALETTE

For this design the main color I choosed was pure white as the background color. The color for the text is a dark grey color. The color for the call to action is bright orange color.

FONT FAMILY

Playfair Display is the type of font I used for the headings in this design, and Open Sans is the type of font I used for call-to-action button and other texts.
